Definition



La balise <html> indique au navigateur qu'il s'agit d'un document HTML. La balise <html> représente la racine d'un document HTML. La balise <html> est le conteneur pour tous les autres éléments HTML (à l'exception de la balise <! DOCTYPE>).










Exemple



<!DOCTYPE HTML> <html> <head>
