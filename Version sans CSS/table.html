Definition



La balise <table> définit une table HTML. Une table HTML comprend l'élément <table> et un ou plusieurs éléments <tr>, <th> et <td>. L'élément <r> définit une ligne de table, l'élément <th> définit un en-tête de table et l'élément <td> définit une cellule de table. Un tableau HTML plus complexe peut également inclure les éléments <caption>, <col>, <colgroup>, <thead>, <tfoot> et <tbody>.

La balise <td> définit une cellule standard dans un tableau HTML.

Un tableau HTML a deux types de cellules: Cellules d'en-tête - contient les informations d'en-tête (créées avec l'élément <th>) Cellules standard - contient des données (créées avec l'élément <td>) Le texte des éléments <th> est en gras et centré par défaut. Le texte dans les éléments <td> est normal et aligné à gauche par défaut.

La balise <tr> définit une ligne dans un tableau HTML. Un élément <tr> contient un ou plusieurs éléments <th> ou <td>.










Exemple



<table> <tr> <th>Mois</th> <th>Economies</th> </tr> <tr> <td>Janvier</td> <td>100€</td> </tr> </table>
