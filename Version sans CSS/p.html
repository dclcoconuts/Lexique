Definition



La balise <p> définit un paragraphe. Les navigateurs ajoutent automatiquement de l'espace (marge) avant et après chaque élément <p>. Les marges peuvent être modifiées avec CSS (avec les propriétés de marge).
The <ul> tag defines an unordered (bulleted) list. Use the <ul> tag together with the <li> tag to create unordered lists.
The <li> tag defines a list item. The <li> tag is used in ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).
La balise <br> insère un seul saut de ligne. La balise <br> est une balise vide, ce qui signifie qu’elle n’a pas de balise de fin.
La balise <hr> définit une rupture thématique dans une page HTML (par exemple, un changement de sujet). L'élément <hr> est utilisé pour séparer le contenu (ou définir un changement) dans une page HTML.
